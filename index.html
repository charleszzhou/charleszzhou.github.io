Projects

Autonomous Submersible Robot

Objective: Develop an autonomous robot for depth measurement and resurfacing. Outcome: Award-winning design with wireless telemetry.

  • ESP32 microcontroller with pressure-based depth detection.
  • Solenoid-driven buoyancy control using a pressurized air bladder.
  • Waterproof PETG housing and modular sensor mount.
Design Process / Validation
  • Design: Iterated 5 prototypes balancing buoyancy and stability.
  • Validation: Calibrated in a 17-ft tank; tested resurfacing reliability.
  • Reflection: Learned waterproofing trade-offs and loop stability tuning.

CubeMars CAN Control Interface

Objective: Build a safe, modular GUI for controlling multiple CubeMars motors. Outcome: Real-time torque, velocity, and position control.

  • Developed Python GUI using python-can and Tkinter.
  • Implemented MIT mode control with heartbeat safety and logging.
  • Daisy-chained CAN H/L lines between AK-series motors.
Extra: Position Control Demo
  • Position control test (YouTube)
  • Validated PID gains and ramp rates for stable performance.

Soft Jumper — Clutch Mechanisms

Objective: Develop energy-release clutches for a soft robotic jumper. Outcome: Achieved 36” jump and consistent pawl engagement.

  • Designed pawl, electromagnetic, and pulley-based clutches.
  • Analyzed timing using high-speed footage and force data.
Extra: Pawl Clutch Test
  • Pawl clutch short demo

Torsional Stiffness Measurement Apparatus

Objective: Create a low-cost apparatus for measuring soft robotic arm stiffness. Outcome: ±5% repeatability; 0.1 Nm/deg resolution.

  • Integrated rotary encoder and load cell with 3D-printed couplers.
  • Characterized torque-angle data using MATLAB and LabVIEW.
