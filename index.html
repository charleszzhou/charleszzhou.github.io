CubeMars Motor Control Interface

System diagram / GUI screenshot placeholder

Objective: Create a Python-based CAN interface for controlling CubeMars brushless motors in MIT mode.

  • Used Seeed Studio USB-CAN analyzer for communication with AK40-10 and AK80-64 motors.
  • Developed a real-time control GUI with heartbeat safety, torque/velocity monitoring, and dropdown motor selection.
  • Optimized code for modular control and decoding feedback packets.

Tools: Python 3.13, python-can, Tkinter GUI, CubeMars CAN protocol.

Code: View on GitHub

Reflection: Gained experience with embedded communication, CAN bus safety, and modular software architecture.

Soft Jumper Robot Clutch Mechanism

Video embed placeholder
Mechanical diagram / test photo placeholder

Objective: Design and evaluate clutch systems enabling rapid energy release for a soft jumping robot.

  • Prototyped electromagnetic, pawl, and string-pulley clutch variations using laser-cut and 3D-printed components.
  • Characterized timing and repeatability across iterations using high-speed video and force measurements.
  • Collaborated in UCSB Hawkes Lab to optimize performance beyond biological limits.

Tools: SolidWorks, Onyx 3D printing, Arduino timing system, FEA simulation.

Reflection: Learned trade-offs between reliability, force multiplication, and release synchronization.

Torsional Stiffness Measurement Apparatus

Video embed placeholder
Experimental setup / graph placeholder

Objective: Build a test rig to measure the torsional stiffness of soft robotic arms with high repeatability.

  • Designed modular fixture system with rotary encoder and torque sensor integration.
  • Fabricated parts with 3D-printed couplings and aluminum baseplate for alignment accuracy.
  • Implemented data acquisition using LabVIEW and MATLAB for torque-angle characterization.

Tools: SolidWorks, FEA validation, 3D printing, MATLAB data analysis.

Reflection: Reinforced understanding of experimental uncertainty, fixture design, and repeatability analysis.
